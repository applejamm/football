# 【技术】PRD 五维权重预测引擎 v1

## 目标与范围

落地 PRD 最小闭环：**五维权重评分 → 3–5 套组合方案 → 历史回测 → 每日模拟**。

## 模块

| 文件 | 职责 |
|---|---|
| `weights.yaml` | 维度权重 30/25/20/15/10、H2H 衰减、方案数量 |
| `predict_lib.py` | 五维评分、胜平负/总进球/比分概率、组合生成 |
| `predict_engine.py` | CLI：读 odds + fundamentals → `prediction_*.json`；合并报告写入 `reports/` |
| `backtest/run_backtest.py` | 30 场种子回测，输出准确率 |
| `daily_simulate.py` | 每日最多 5 场 → `simulation/` |
| `scripts/prefetch.sh` | 赛前自动抓取 + 可选 predict |

## 五维 v1 数据映射

| 维度 | v1 数据来源 | 缺失处理 |
|---|---|---|
| 硬实力 | 近 5/10 场形式、胜率、场均进失球 | 半权重 + 50 分 |
| 人员 | injuries / roster（国家队多为暂无） | 50 分 + 标记 |
| 大赛 | 新闻关键词 + 共识大小球 | 50 分 |
| H2H | 近 3 次 + 时间衰减 | 半权重 |
| 市场 | 体彩 1X2 或 DraftKings | 仅基本面推断 |
