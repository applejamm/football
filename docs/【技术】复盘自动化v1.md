## 目标

赛后一条命令完成：**注单结算 → tracking 回填 → decision MD/HTML 赛后态 → 虚拟跟踪联动**，把「赛前决策」闭环到「可核对盈亏」。

## 模块

| 文件 | 职责 |
|---|---|
| `replay_lib.py` | 解析行动卡、结算、patch MD/HTML、tracking 行匹配 |
| `replay_decision.py` | CLI（`--code` / `--decision` + `--scores`） |
| `replay/runs/*.json` | 结算归档 |
| `tests/test_replay_decision.py` | 解析 + 命中/未中 + hook 回填 |

## 输入

| 参数 | 说明 |
|---|---|
| `decision_*.md` | 行动卡表格（注 ID / 场次 / 玩法 / 选择 / 赔率 / 金额） |
| `--scores` | `周二017:2:0` 逗号分隔 |
| `--scan`（可选） | 联动虚拟跟踪；缺省取 `validation/drafts/scan_<code>*.json` 最新 |

## 产出

| 目标 | 变化 |
|---|---|
| `decision_*.md` | §4 复盘 hook 填比分 / 命中 / 收益 |
| `decision_*.html` | `body.post-settled`、hero 绿/红边、结局条切实际、复盘表 |
| `tracking.md` | 注级明细匹配「期号 + 注 ID」行回填 |
| `tracking.md` 虚拟段 | 有 scan 时自动刷新（同 T-5-5） |

## 命令

```bash
cd /Users/CursorProject/football

python3 replay_decision.py --code 260616 --scores "周二017:2:0"

python3 replay_decision.py \\
  --decision decision_260616_match017_FUNNEL_REGEN.md \\
  --scores "周二017:2:0" \\
  --scan validation/drafts/scan_260616_demo.json
```
